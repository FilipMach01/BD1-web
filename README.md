# BD Kutná Hora – Strakoschova

Webový portál Bytového družstva Kutná Hora Strakoschova.

## Stránky

- **index.html** – Úvodní strana
- **nastenka.html** – Nástěnka (oznámení, schůze, dokumenty)
- **kontakty.html** – Kontakty na správu a vedení
- **zakladni-info.html** – Základní informace o družstvu
- **admin.html** – Administrační přihlášení

## Technologie

- HTML5 + Tailwind CSS (CDN)
- Google Fonts (Public Sans)
- Material Symbols (ikony)
- Žádný build krok – statické HTML soubory
